Convex는 다음과 같은 값 타입을 지원합니다:

<div className="convex-full-width">

| Convex 타입 | TS/JS 타입                                                                                                                   | <div style={{width: "11em"}}>사용 예시</div> | [인자 검증](/functions/validation)과 [스키마](/database/schemas)를 위한 Validator | [내보내기](/database/import-export)를 위한 `json` 형식 | 참고                                                                                                                                                                                                                                                                                                                                                  |
| ----------- | ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ | ------------------------------------------------------------------------------------------- | --------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Id          | [Id](/database/document-ids) ([string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)) | `doc._id`                                        | `v.id(tableName)`                                                                           | string                                              | [문서 ID](/database/document-ids)를 참조하세요.                                                                                                                                                                                                                                            |
| Null        | [null](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#null_type)                                    | `null`                                           | `v.null()`                                                                                  | null                                                | JavaScript의 `undefined`는 유효한 Convex 값이 아닙니다. `undefined`를 반환하거나 반환하지 않는 함수는 클라이언트에서 호출될 때 `null`을 반환합니다. 대신 `null`을 사용하세요.                                                                                                                                                                              |
| Int64       | [bigint](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#bigint_type)                                | `3n`                                             | `v.int64()`                                                                                 | string (base10)                                     | Int64는 -2^63과 2^63-1 사이의 BigInt만 지원합니다. Convex는 [대부분의 최신 브라우저](https://caniuse.com/bigint)에서 `bigint`를 지원합니다.                                                                                                                                                                                                                 |
| Float64     | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#number_type)                                | `3.1`                                            | `v.number()`                                                                                | number / string                                     | Convex는 모든 IEEE-754 배정밀도 부동소수점 숫자(NaN 포함)를 지원합니다. Inf와 NaN은 JSON으로 문자열로 직렬화됩니다.                                                                                                                                                                                                                       |
| Boolean     | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#boolean_type)                              | `true`                                           | `v.boolean()`                                                                               | bool                                                |
| String      | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#string_type)                                | `"abc"`                                          | `v.string()`                                                                                | string                                              | 문자열은 UTF-8로 저장되며 유효한 유니코드 시퀀스여야 합니다. UTF-8로 인코딩했을 때 1MB 전체 크기 제한보다 작아야 합니다.                                                                                                                                                                                                          |
| Bytes       | [ArrayBuffer](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer)                  | `new ArrayBuffer(8)`                             | `v.bytes()`                                                                                 | string (base64)                                     | Convex는 `ArrayBuffer`로 전달되는 일급 바이트 문자열을 지원합니다. 바이트 문자열은 Convex 타입의 1MB 전체 크기 제한보다 작아야 합니다.                                                                                                                                                                                                      |
| Array       | [Array](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array)                              | `[1, 3.2, "abc"]`                                | `v.array(values)`                                                                           | array                                               | 배열은 최대 8192개의 값을 가질 수 있습니다.                                                                                                                                                                                                                                                                                                                   |
| Object      | [Object](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Data_structures#objects)                                    | `{a: "abc"}`                                     | `v.object({property: value})`                                                               | object                                              | Convex는 "순수 JavaScript 객체"(커스텀 프로토타입이 없는 객체)만 지원합니다. Convex는 모든 [열거 가능한 속성](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Enumerability_and_ownership_of_properties)을 포함합니다. 객체는 최대 1024개의 항목을 가질 수 있습니다. 필드명은 비어있지 않아야 하며 "$" 또는 "\_"로 시작할 수 없습니다. |
| Record      | [Record](https://www.typescriptlang.org/docs/handbook/utility-types.html#recordkeys-type)                                    | `{"a": "1", "b": "2"}`                           | `v.record(keys, values)`                                                                    | object                                              | Record는 런타임에 객체이지만 동적 키를 가질 수 있습니다. 키는 ASCII 문자만 포함해야 하며, 비어있지 않아야 하고, "$" 또는 "\_"로 시작할 수 없습니다.                                                                                                                                                                                                               |
