[Convex Auth](https://labs.convex.dev/auth)는 Convex 백엔드 내에서 직접 인증을 구현하기 위한 라이브러리입니다. 이를 통해 인증 서비스나 호스팅 서버 없이도 사용자를 인증할 수 있습니다. Convex Auth는 현재 CDN에서 제공되는 클라이언트 측 React 웹 앱과 React Native 모바일 앱을 지원합니다.

**예제:** [라이브 데모](https://labs.convex.dev/auth-example)
([소스](https://github.com/get-convex/convex-auth-example))

<BetaAdmonition feature="Convex Auth" verb="is" />

[Next.js에서의 인증](https://labs.convex.dev/auth/authz/nextjs) 서버 컴포넌트, API 라우트, 미들웨어, SSR 등에 대한 지원은 활발히 개발 중입니다. 이 실험적 지원을 테스트하는 데 도움을 주고 싶다면 [Discord에서 어떻게 진행되는지 알려주세요](https://convex.dev/community).

## 시작하기

Convex와 Convex Auth로 처음부터 새 프로젝트를 시작하려면 다음을 실행하세요:

```sh
npm create convex@latest
```

그리고 `React (Vite)`와 `Convex Auth`를 선택하세요.

---

기존 프로젝트에 Convex Auth를 추가하려면 전체 [설정 가이드](https://labs.convex.dev/auth/setup)를 따르세요.

## 개요

Convex Auth를 사용하면 다음 인증 방법을 구현할 수 있습니다:

1. 매직 링크 & OTP - 이메일로 링크 또는 코드 전송
2. OAuth - GitHub / Google / Apple 등으로 로그인
3. 비밀번호 - 비밀번호 재설정 플로우 및 선택적 이메일 인증 포함

라이브러리는 UI 컴포넌트와 함께 제공되지 않지만, 문서와 예제 리포지토리에서 코드를 복사하여 React에서 UI를 빠르게 구축할 수 있습니다.

[Convex Auth 문서](https://labs.convex.dev/auth)에서 자세히 알아보세요.
