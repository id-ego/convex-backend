# Convex JavaScript 클라이언트

Convex 애플리케이션은 [`fetch`](https://developer.mozilla.org/en-US/docs/Web/API/fetch) 또는
[`WebSocket`](https://developer.mozilla.org/en-US/docs/Web/API/WebSocket)을 구현하는 Node.js 또는 모든 JavaScript 런타임에서 액세스할 수 있습니다. 반응형 [Convex 클라이언트](/api/classes/browser.ConvexClient)는 웹 애플리케이션과 장기 실행 Node.js 서버가 Convex 쿼리에 대한 업데이트를 구독할 수 있게 하며, [Convex HTTP 클라이언트](/api/classes/browser.ConvexHttpClient)는 일반적으로 서버 측 렌더링, 마이그레이션, 관리 스크립트 및 서버리스 함수에서 단일 시점의 쿼리를 실행하는 데 사용됩니다.

React를 사용하는 경우 [React](/client/react.mdx)에 설명된 전용
[`ConvexReactClient`](/api/classes/browser.ConvexClient)를 참조하세요.

## Convex 클라이언트

[`ConvexClient`](/api/classes/browser.ConvexClient)는 Node.js 및 WebSocket을 지원하는 모든 JavaScript 환경에서 쿼리에 대한 구독을 제공합니다.

Convex 클라이언트는 오픈 소스이며
[GitHub](https://github.com/get-convex/convex-js)에서 사용 가능합니다.

시작하려면 [스크립트 태그 빠른 시작](/quickstart/script-tag.mdx)을 참조하세요.
