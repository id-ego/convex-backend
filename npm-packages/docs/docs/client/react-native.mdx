[React Native](https://reactnative.dev/)에서 Convex를 사용하려면
[Convex React 클라이언트 라이브러리](/client/react.mdx)를 사용하세요.

React Native에 특별히 필요한 다른 구성은 [React Native 빠른 시작](/quickstart/react-native.mdx)을 따르세요.

작동하는
[Convex React Native 데모](https://github.com/get-convex/convex-demos/tree/main/react-native)를 클론할 수도 있습니다.
