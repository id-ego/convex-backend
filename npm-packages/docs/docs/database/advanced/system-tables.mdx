  "예약된 함수와 파일 스토리지 정보를 포함한 Convex 내장 기능의 메타데이터에 시스템 테이블을 통해 액세스합니다."
---

시스템 테이블은 내장 Convex 기능의 메타데이터에 대한 읽기 전용 액세스를 제공합니다.
현재 두 개의 시스템 테이블이 노출되어 있습니다:

- `"_scheduled_functions"` 테이블은
  [예약된 함수](/scheduling/scheduled-functions.mdx#retrieving-scheduled-function-status)의 메타데이터를 포함합니다
- `"_storage"` 테이블은
  [저장된 파일](/file-storage/file-metadata.mdx)의 메타데이터를 포함합니다

`db.system.get` 및 `db.system.query` 메서드를 사용하여 시스템 테이블에서 데이터를 읽을 수 있으며, 이는 표준 `db.get` 및 `db.query` 메서드와 동일하게 작동합니다. 시스템 테이블에서 읽는 쿼리는 다른 모든 테이블에서 읽는 쿼리와 마찬가지로 반응형이고 실시간이며, 단일 쿼리에서 읽기에는 너무 많은 경우에도 페이지네이션을 사용하여 모든 문서를 열거할 수 있습니다.
