Convex 배포의 공개 인터페이스는 convex 폴더의 파일에 정의된 함수들로 정의됩니다.

모든 Convex 배포의 공개 HTTP 엔드포인트는 [HTTP Actions](docs/functions/http-actions.mdx)로 정의된 사용자 정의 HTTP 엔드포인트와 정적 [공개 HTTP API](/docs/http-api/index.md)로 구성됩니다.

배포는 또한 해당 배포의 관리자만 사용할 수 있는 비공개 엔드포인트를 제공합니다:

- [Streaming export API](/docs/streaming-export-api.mdx)
- [Streaming import API](/docs/streaming-import-api.mdx)
- [Platform APIs](/docs/deployment-platform-api.mdx)
