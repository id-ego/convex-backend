파일 스토리지를 사용하면 앱에서 파일 업로드를 쉽게 구현하고, 서드파티 API에서 파일을 저장하고 전송하며, 사용자에게 동적 파일을 제공할 수 있습니다. 모든 파일 유형이 지원됩니다.

- Convex에 파일을 저장하고 데이터베이스 문서에서 참조하려면 파일을 [업로드](/file-storage/upload-files.mdx)하세요
- 서드파티 API에서 생성되거나 가져온 파일을 [저장](/file-storage/store-files.mdx)하세요
- URL을 통해 파일을 [제공](/file-storage/serve-files.mdx)하세요
- Convex에 저장된 파일을 [삭제](/file-storage/delete-files.mdx)하세요
- 파일 [메타데이터](/file-storage/file-metadata.mdx)에 액세스하세요

[대시보드](/dashboard/deployments/files.md)에서 저장된 파일을 관리할 수 있습니다.

**예제:**
[HTTP Actions를 사용한 파일 스토리지](https://github.com/get-convex/convex-demos/tree/main/file-storage-with-http),
[Queries 및 Mutations를 사용한 파일 스토리지](https://github.com/get-convex/convex-demos/tree/main/file-storage)
