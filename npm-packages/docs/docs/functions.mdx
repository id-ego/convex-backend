함수는 백엔드에서 실행되며 JavaScript(또는 TypeScript)로 작성됩니다. 이들은 [클라이언트 라이브러리](/client/react.mdx)를 통해 액세스되는 API로 자동으로 제공됩니다. Convex 백엔드에서 수행하는 모든 작업은 함수에서 시작됩니다.

세 가지 유형의 함수가 있습니다:

- [쿼리](/functions/query-functions.mdx)는 Convex 데이터베이스에서 데이터를 읽으며 자동으로 캐시되고 구독 가능합니다(실시간, 반응형).
- [뮤테이션](/functions/mutation-functions.mdx)은 데이터베이스에 데이터를 작성하고 트랜잭션으로 실행됩니다.
- [액션](/functions/actions.mdx)은 OpenAI, Stripe, Twilio 또는 앱 작동에 필요한 기타 서비스나 API를 호출할 수 있습니다.

웹훅이나 커스텀 클라이언트에서 함수를 호출하려는 경우 [HTTP 액션](/functions/http-actions.mdx)을 구축할 수도 있습니다.

다음은 세 가지 유형의 Convex 함수와 그 기능에 대한 개요입니다:

|                            | 쿼리 | 뮤테이션 | 액션 |
| -------------------------- | ------- | --------- | ------- |
| 데이터베이스 액세스            | 예     | 예       | 아니오      |
| 트랜잭션              | 예     | 예       | 아니오      |
| 캐시됨                     | 예     | 아니오        | 아니오      |
| 실시간 업데이트          | 예     | 아니오        | 아니오      |
| 외부 API 호출 (fetch) | 아니오      | 아니오        | 예     |
