풀스택 웹 앱을 게시하는 가장 쉬운 방법은 [Vercel](https://vercel.com) 또는 [Netlify](https://netlify.com)와 같은 호스팅 제공자를 사용하는 것입니다.

Vercel과 Netlify는 모두 Git과 통합되어 새 리비전이 푸시될 때마다 코드를 배포합니다. 앱을 호스팅하려면:

1. 모든 파일을 커밋하고 [GitHub](https://github.com/), [GitLab](https://gitlab.com/) 또는 [Bitbucket](https://bitbucket.org/)과 같은 원하는 Git 호스팅 제공자에 푸시하세요.

2. 아래의 적절한 가이드를 따르세요.

Netlify 또는 Vercel을 사용하지 않는 경우 커스텀 호스팅 가이드를 따를 수 있습니다.
