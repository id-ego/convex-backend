Tokio와 함께 Rust 앱에서 Convex 데이터를 쿼리하는 방법을 알아보세요.

<StepByStep>
  <Step title="Cargo 프로젝트 생성하기">
    새 Cargo 프로젝트를 생성하세요.

    ```sh
    cargo new my_app
    cd my_app
    ```

  </Step>
  <Step title="Convex 클라이언트 및 서버 라이브러리 설치하기">
    시작하려면 백엔드를 작성할 수 있게 해주는 `convex` npm 패키지를 설치하세요.

    또한 `convex` Rust 클라이언트 라이브러리, `tokio` 런타임, 그리고 `.env` 파일 작업을 위한 `dotenvy`를 설치하세요.

    ```sh
    npm init -y && npm install convex && cargo add convex tokio dotenvy
    ```

  </Step>
  <Step title="Convex 개발 배포 설정하기">
    다음으로, `npx convex dev`를 실행하세요. 이것은 GitHub로 로그인하고, 프로젝트를 생성하고, 프로덕션 및 배포 URL을 저장하라는 메시지를 표시합니다.

    또한 백엔드 API 함수를 작성할 수 있는 `convex/` 폴더를 생성합니다. `dev` 명령은 계속 실행되면서 함수를 클라우드의 개발 배포와 동기화합니다.


    ```sh
    npx convex dev
    ```

  </Step>

  <Step title="데이터베이스용 샘플 데이터 생성하기">
    새 터미널 창에서 샘플 데이터가 포함된 `sampleData.jsonl` 파일을 생성하세요.

    <Snippet
      source={sampleData}
      title="sampleData.jsonl"
    />

  </Step>

  <Step title="데이터베이스에 샘플 데이터 추가하기">
    이제 프로젝트가 준비되었으므로 `import` 명령을 사용하여 샘플 데이터가 포함된 `tasks` 테이블을 Convex 데이터베이스에 추가하세요.

    ```
    npx convex import --table tasks sampleData.jsonl
    ```

  </Step>

  <Step title="데이터베이스 쿼리 노출하기">
    `convex/` 폴더에 데이터를 로드하는 쿼리 함수가 포함된 새 파일 `tasks.js`를 추가하세요.

    이 파일에서 쿼리 함수를 내보내면 파일과 내보내기 이름으로 명명된 API 함수 `"tasks:get"`이 선언됩니다.

    <Snippet
      source={tasks}
      title="convex/tasks.js"
    />

  </Step>

  <Step title="앱을 백엔드에 연결하기">
    `src/main.rs` 파일에서 `ConvexClient`를 생성하고 이를 사용하여 `"tasks:get"` API에서 데이터를 가져오세요.

    <Snippet
      source={main}
      title="src/main.rs"
    />

  </Step>

  <Step title="앱 실행하기">
      앱을 실행하고 직렬화된 작업 목록을 확인하세요.

      ```sh
      cargo run
      ```

  </Step>

</StepByStep>

전체 [Rust 문서](https://docs.rs/convex/latest/convex/)를 참조하세요.
