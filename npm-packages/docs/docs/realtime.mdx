Convex는 자동으로 실시간으로 작동합니다! 앱에서 이미 [쿼리 함수](./functions/query-functions),
[데이터베이스](./database), [클라이언트 라이브러리](./client/react/)를 사용하고 있다면
특별한 작업을 할 필요가 없습니다.
Convex는 데이터베이스 변경 사항을 포함하여 쿼리 함수의 종속성을 추적하고,
클라이언트 라이브러리에서 구독을 트리거합니다.

<div align="center">
  ![Convex is automatically reactive and realtime](/img/realtime.gif)
</div>

고도로 인터랙티브한 앱을 쉽게 구축하는 것 외에도, Convex의 실시간 아키텍처에는
다음과 같은 다른 이점들이 있습니다:

## 자동 캐싱

Convex는 쿼리 함수의 결과를 자동으로 캐싱하여 향후 호출 시 캐시에서 읽기만 하면 됩니다.
데이터가 변경되면 캐시가 업데이트됩니다.
캐시된 읽기에 대해서는 데이터베이스 대역폭 요금이 부과되지 않습니다.

이를 위해 별도의 작업이나 기록 관리가 필요하지 않습니다.

## 앱 전체에서 일관된 데이터

모든 클라이언트 구독은 데이터베이스의 동일한 스냅샷으로 동시에 업데이트됩니다.
앱은 항상 데이터의 가장 일관된 뷰를 표시합니다.

이를 통해 장바구니의 항목 수를 늘렸지만 품절된 항목을 표시하지 않는 것과 같은
버그를 방지할 수 있습니다.
