셀프 호스팅에 관심이 있다면, 자체 서버에서 Convex 백엔드를 실행할 수 있습니다. 셀프 호스팅 Convex는
[오픈 소스 백엔드](https://github.com/get-convex/convex-backend)를 실행하며, 클라우드 서비스가 사용하는 것과 동일한 완전히 최신 코드를 포함합니다.

셀프 호스팅을 시작하려면 셀프 호스팅 가이드를 따르세요:

셀프 호스팅 지원을 위해 [Discord 커뮤니티](https://convex.dev/community)의 `#self-hosted` 채널에 참여하세요.

셀프 호스팅은 모든 사람을 위한 것은 아닙니다. 보다 관리가 쉬운 솔루션을 찾고 있다면,
[Convex 호스팅 제품](https://convex.dev/pricing)을 사용하는 것을 권장합니다.

## 오픈 소스 Convex 백엔드

백엔드의 대부분은 Rust로 작성되었으며, 서버 측 함수 환경을 지원하는 TypeScript가 상당 부분 포함되어 있습니다.

Convex의 오픈 소스화에 대한 자세한 내용은
[공지](https://news.convex.dev/convex-goes-open-source/)와
[소프트웨어 엔지니어링 데일리 팟캐스트](https://softwareengineeringdaily.com/2024/03/20/going-open-source-at-convex-with-james-cowling/)에서 확인할 수 있습니다.
[FSL Apache 2.0 라이선스](https://fsl.software/)로 배포됩니다.

## 기타 Convex 오픈 소스 프로젝트

Convex 백엔드, 클라이언트 라이브러리, 대시보드 및 CLI는 모두 오픈 소스입니다.
[Convex GitHub 페이지](https://github.com/get-convex)에서 모든 것을 탐색할 수 있습니다.

### Convex 클라이언트

모든 Convex 클라이언트는 오픈 소스입니다.

- [Convex JavaScript/TypeScript 클라이언트 & CLI](https://github.com/get-convex/convex-js)
- [Convex Python 클라이언트](https://github.com/get-convex/convex-py)
- [Convex Rust 클라이언트](https://github.com/get-convex/convex-rs)

### 그 외 훨씬 더 많은 것들

Convex는 또한
[헬퍼](https://github.com/get-convex/convex-helpers),
[템플릿](https://github.com/orgs/get-convex/repositories?type=all&q=template),
[데모](https://github.com/get-convex/convex-demos),
[테스트 하네스](https://github.com/get-convex/convex-test) 등을 포함한 다양한 유용한 프로젝트를 오픈 소스로 공개합니다. 모든 공개 리포지토리의 전체 목록은
[GitHub](https://github.com/orgs/get-convex/repositories?type=all)에서 확인하세요.
