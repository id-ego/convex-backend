Convex는 JS에서 실행되는 자동화된 테스트나 실제 백엔드에 대한 테스트, 그리고 개발, 프리뷰, 스테이징 환경에서의 수동 테스트를 통해 앱을 쉽게 테스트할 수 있게 합니다.

## 자동화된 테스트

### `convex-test` 라이브러리

[`convex-test` 라이브러리 사용](/testing/convex-test.mdx)을 통해 뛰어난 Vitest 테스트 프레임워크를 통해 JS로 함수를 테스트할 수 있습니다.

### 실제 백엔드에 대한 테스트

Convex 오픈소스 빌드를 사용하면 실제 [로컬 Convex 백엔드](/testing/convex-backend.mdx)에서 실행되는 모든 백엔드 로직을 테스트할 수 있습니다.

### CI에서 테스트 설정

제어된 환경에서 앱을 지속적으로 테스트하는 것이 좋습니다. 사용하는 자동화 방법에 관계없이 [GitHub Actions](/testing/ci.mdx)로 쉽게 실행할 수 있습니다.

## 수동 테스트

### 개발 환경에서 함수 실행

개발 환경에서 함수를 수동으로 실행하여 작동 여부를 빠르게 확인합니다:

- [명령줄에서 함수 실행](/cli.md#run-convex-functions)
- [대시보드에서 함수 실행](/dashboard/deployments/functions.md#running-functions)

### 프리뷰 배포

[프리뷰 배포 사용](/production/hosting/preview-deployments.mdx)을 통해 진행 중인 기능에 대해 팀으로부터 조기 피드백을 받습니다.

### 스테이징 환경

테스트할 스테이징 환경으로 별도의 프로젝트를 설정할 수 있습니다. [앱을 프로덕션에 배포하기](/production.mdx#staging-environment)를 참조하세요.
